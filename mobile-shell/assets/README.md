# Mobile App Assets

Source images for generating iOS and Android app icons and splash screens.

Everything here derives from one master: `dragonhub_icon.png`, the DragonHub
dragon-and-schoolhouse logo as a square PNG with a genuine transparent
background. Archived past logos live in
`claude_code_instructions/dragon_hub_logos/`.

## Regenerating after a logo change

```bash
# 1. Drop the new square, transparent master in as dragonhub_icon.png, then:
./mobile-shell/assets/prepare-icon.sh   # derives icon / foreground / background
# 2. Splashes are hand-made — see "Splash screens" below.
# 3. Fan out to the native projects:
npm run mobile:assets
```

`npm run mobile:assets` runs `@capacitor/assets generate --assetPath
mobile-shell/assets` and writes:

- `ios/App/App/Assets.xcassets/AppIcon.appiconset/*`
- `ios/App/App/Assets.xcassets/Splash.imageset/*`
- `android/app/src/main/res/mipmap-*/ic_launcher*`
- `android/app/src/main/res/drawable*/splash.png`
- `icons/*.webp` at the repo root — a side effect we don't use. The PWA icons
  the app actually serves are the PNGs in `public/icons/`, wired up through
  `public/manifest.webmanifest`. The root `icons/` directory is gitignored.

## Files

| File | Size | Purpose |
| --- | --- | --- |
| `dragonhub_icon.png` | square PNG, transparent | **The master.** Everything below is derived from it. |
| `icon.png` | **1024×1024** PNG, no alpha | iOS app icon and the Android legacy icon. Generated by `prepare-icon.sh` — logo on white at 94% scale. iOS rejects icons that carry an alpha channel. |
| `icon-foreground.png` | **1024×1024** PNG, transparent | Android adaptive icon foreground. Generated at 68% scale so the launcher's circle/squircle mask can't clip the dragon. |
| `icon-background.png` | **1024×1024** PNG, solid | Android adaptive icon background. Solid white, matching `icon.png`. |
| `splash.png` | **2732×2732** PNG | Light-mode launch splash: logo centered on white. |
| `splash-dark.png` | **2732×2732** PNG | Dark-mode launch splash: logo centered on `dragon-blue-900`. |

## Splash screens

Both splashes are the master logo at ~1100px centered on a 2732×2732 canvas:

```bash
cd mobile-shell/assets
magick dragonhub_icon.png -resize 1100x1100 -background white \
  -gravity center -extent 2732x2732 -alpha remove -alpha off splash.png

magick dragonhub_icon.png -resize 1100x1100 -background "#0f172a" \
  -gravity center -extent 2732x2732 -alpha remove -alpha off splash-dark.png
```

## Color reference

DragonHub brand colors, from `src/app/globals.css`:

- `--color-dragon-blue-900: #0f172a` — dark splash background, public landing page
- `--color-dragon-blue-500: #1e40af` — primary, and the PWA `theme_color`
- `--color-dragon-gold-400: #facc15` — accent

White sits behind the icon rather than a brand color because the logo's own
outline is a near-navy that disappears against `dragon-blue-900`.
